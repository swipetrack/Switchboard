Class DynamicQueue<T>

Namespace
Switchboard
Assembly
SwitchboardCore.dll

Stores objects of a specific type in a queue. Objects are kept in the order of first in, first out. The internal memory used to store these objects grows and shrinks automatically as objects are added and removed. Unused memory is returned to the SharedObjectArrayPool, so no garbage collection is necessary.
