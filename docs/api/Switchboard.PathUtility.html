GetFileName(string)

Uses a static Dictionary<TKey, TValue> to cache file names parsed from a given path string. The method is thread safe. It calls GetFileName(string) and stores the result. So, if the same filePath is evaluated multiple times, the value is simply retrieved from the cache.
